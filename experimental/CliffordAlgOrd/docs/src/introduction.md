# Introduction

This project provides functionality for constructing and working with Clifford algebras associated with quadratic spaces.
Currently, it supports base rings such as the rationals, algebraic number fields, and their respective rings of integers.

## Content

This project is structured as follows:

- The section [Clifford algebras](@ref cliffordalgebras) contains functionality for Clifford algebras over fields (e.g., `QQ` or other algebraic number fields).
- The section [Clifford orders](@ref cliffordorders) contains functionality for Clifford orders (i.e. orders within Clifford algebras over the rings of integers, such as `ZZ` or ``\mathcal{O}_K``).
