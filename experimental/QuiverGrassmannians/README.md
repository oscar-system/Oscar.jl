
##Aims
We compute coordinate rings of Quiver Grassmannians corresponding to a fixed quiver, ambient dimension vector,  subspace dimension vector, and collection of linear maps.
The input data consists of: 
-Quiver Q (directed graph) 
-A vector N of ambient dimensions for each vertex of Q 
-A vector D of subspace dimensions for each vertex of Q
-A vector E of linear maps for each edge
Output: Coordinate ring of moduli space of realizations of the given quiver representation.
